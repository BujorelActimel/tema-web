# Currency Converter

Convertor de valuta

## Cum sa rulezi proiectul local

1. Cloneaza repository-ul: (sau descarca-l manual)
```bash
git clone https://github.com/BujorelActimel/tema-web.git
cd tema-web
```

2. Instaleaza dependentele:
```bash
npm install
```

3. Porneste serverul:
```bash
node server.js
```

4. Deschide browserul si navigheaza la:
```
http://localhost:3000
```
